# Code for "On-demand phase-field modeling: Three-dimensional Landau energy for HfO2 through machine learning"

This repository contains the code used for the analyses in the manuscript:

**On-demand phase-field modeling: Three-dimensional Landau energy for HfO2 through machine learning**  
Yusuke Tamura, Kairi Masuda, Shin Kiyohara, Yu kumagai  
Manuscript under review.

## Usage

The main code is provided as a Jupyter Notebook:

```text
notebooks/On_demand_phase_field_HfO2.ipynb
```

The notebook is organized in the following order:

1. Construction of the training dataset
2. Training of the NN-based Landau energy
3. Phase-field simulation

Please open the notebook with Jupyter Notebook or JupyterLab and run the cells from top to bottom.

The datasets, training parameters, and CIF files for HfO2 used in the manuscript are provided in the `data/` directory.
